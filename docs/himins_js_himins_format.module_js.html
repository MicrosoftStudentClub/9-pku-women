Methods

(inner) formatText(context, template, indent, width, style) → {String}

Transforms context and template into text. Waps text to fit column specified by indent and columnWidth. (See Mustache documentation for context and template usage.) Styles text according to style (See Colors documenation for style.) Returns the rendered text.
